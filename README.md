# Digital Logic Design Primitives

This repository contains educational implementations of fundamental combinational and sequential digital circuits developed as part of my exploration into digital system design and future Electronic Design Automation (EDA) research.

## Implemented Modules

### Combinational Logic
- Half Adder
- Full Adder
- Ripple Carry Adder
- Multiplexer (2:1)
- Decoder (3-to-8)

### Sequential Logic
- D Flip-Flop (behavioral model)
- 4-bit Counter
- Finite State Machine (traffic light controller)

## Motivation

These implementations reinforce concepts from my Logic Design coursework, including Boolean algebra, Karnaugh minimization, and sequential circuit design. This repository represents an early step toward deeper work in RTL design and EDA workflows.
